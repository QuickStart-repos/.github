# Contributing to QuickStart-repos `.github`

Thank you for helping improve this repository! Contributions make it easier for everyone to request new templates and report issues.

---

## How to Contribute

I accept contributions through **forks and pull requests**:

1. Fork this repository.  
2. Create a new branch for your changes.  
3. Make your changes, ensuring they follow the current structure.  
4. Submit a pull request describing your changes clearly.  

> For contributions outside of documentation or bug fixes (e.g., adding new templates or workflows), please first open an issue to discuss your suggestion before creating a PR.

---

## What You Can Contribute

- **Documentation updates** – improve clarity, fix typos, or add examples.  
- **Bug fixes** – correct errors in existing templates or workflows.  
- **Suggestions** – propose new templates or workflow improvements via issues first.

All other contributions should be discussed via an issue before submitting a PR.

---

## Testing

Please ensure that any workflow templates or issue templates you submit **function correctly** before submitting your PR.
